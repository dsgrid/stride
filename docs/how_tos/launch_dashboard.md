(launch-dashboard)=
# Launch the Dashboard

View project results interactively in a web browser.

## Start the Dashboard

```{eval-rst}

.. code-block:: console

   $ stride view my_project
```

This opens an interactive dashboard where you can:

- Browse scenarios and data tables
- Create visualizations
- Compare results across geographies and years

## Specify a Port

```{eval-rst}

.. code-block:: console

   $ stride view my_project --port 8080
```

## View Without a Project

Launch the dashboard to create or load a project:

```{eval-rst}

.. code-block:: console

   $ stride view
```
